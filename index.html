プライバシーポリシー — Etudely（楽器練習コックピット拡張）

最終更新: 2026-06-14

概要

Etudely（以下「本拡張」）は、利用者のプライバシーを最優先に設計されています。本拡張は、利用者の個人情報や利用データを収集・送信・共有しません。

取得・保存する情報

本拡張が保存するのは、利用者がブラウザ上で作成した練習設定のみです。

  • ループ区間（セクション名・開始/終了時間）
  • 再生速度・ピッチの設定
  • 上記をひも付ける動画 ID

これらは利用者のブラウザ内（IndexedDB）にローカル保存され、利用者の端末から外部へ送信されることはありません。本拡張は外部サーバーを持ちません。

取得しない情報

  • 氏名・メールアドレス等の個人を特定する情報
  • 閲覧履歴・検索履歴
  • 位置情報・連絡先
  • 認証情報・支払い情報

本拡張はアクセス解析・トラッキング・広告の仕組みを一切含みません。

動画について

本拡張は再生中の動画を操作（再生位置の移動・速度変更・音声のピッチ補正）するだけで、動画や音声のダウンロード・保存・再配布は行いません。

権限

本拡張は YouTube の視聴ページ（https://www.youtube.com/watch*）でのみ動作します。その範囲外のページやデータにはアクセスしません。

データの削除

保存された練習設定は、ブラウザのサイトデータ（YouTube）を消去するか、拡張機能を削除することでいつでも削除できます。

変更

本ポリシーを更新する場合は、本ページの「最終更新」を改訂します。

お問い合わせ

本拡張に関するお問い合わせは、Chrome Web Store の本拡張の掲載ページ（デベロッパー連絡先／サポート欄）からお願いします。
